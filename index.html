Economic Intelligence Dashboard
36 live economic indicators from the Federal Reserve Bank of St. Louis — covering inflation, labor, GDP, interest rates, housing & credit markets.
🚀 Quick Start
1
Get your free API key at fred.stlouisfed.org/docs/api/api_key.html
2
Paste your 32-character key in the top-right field and click Load Data
3
All 36 charts load live. Use the sidebar to navigate categories.
💡 Running Locally
A
Open terminal in this file's folder → run python -m http.server 8080 → open localhost:8080
B
Or deploy to Vercel / GitHub Pages for permanent hosting with HTTPS
